Tutorial

So, you want to add custom html code to a CTLS textbox, here's how. First, go to a text box and click "Insert Link" or CTRL+K. In the URL section type "<" then paste your code script/style headers will not work, but you can embed them. Then in the text tab type #/eContent if you want it to be red, or nothing at all. Press Insert. Click the < then press "Unlink" then remove the < at the start. Done!
