Tutorial

So, you want to add custom HTML code to a CTLS textbox? Here's how:

  1. Go to a text box and click Insert Link or press CTRL+K.
  2. In the URL section type "<", then paste your code (note: script/style headers will not work, but you can embed them).
  3. In the text tab, type #/eContent if you want it to be red, or leave it blank.
  4. Press Insert.
  5. Click the "<", then press Unlink, and remove the "<" at the start.
  6. Done!
